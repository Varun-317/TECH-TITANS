## üì¶ Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Varun-317/TECH-TITANS.git
   cd TECH-TITANS/TECH_TITAN
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server (Lightning Fast ‚ö°):
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.
